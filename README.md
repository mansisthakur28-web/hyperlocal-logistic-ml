# hyperlocal-logistic-ml
An AI/ML-based hyperlocal logistics system that predicts delivery delay risk using historical route data, traffic intensity, and weather conditions. The platform enables proactive route planning, improved fleet efficiency, and cost optimization, with the ML model integrated into backend APIs and visualized through a real-time dashboard.
